# Intune Naming Tool

Static GitHub Pages version of the Intune Naming Tool.

## Naming convention

`Prefix-Type-Target-OS-Name/Goal`

Example:

`JV-CP-D-Win-OneDrive`

## Files

- `index.html`
- `style.css`
- `app.js`
- `LICENSE`

No backend and no config file are required for GitHub Pages.
